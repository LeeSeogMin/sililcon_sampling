### 1. 연도
데이터는 2003-2023년 KGSS 누적 자료로 구성되어 있으며, 분포는 다음과 같은 연도 단위로 제공됩니다:
- 03-07 (2003-2007년 합계)
- 2008
- 2009
- 2010
- 2011
- 2012
- 2013
- 2014
- 2016
- 2018
- 2021
- 2023
- 전체 (2003-2023 합계)

모든 변수에서 연도별 분포는 퍼센트(%)로 표시되며, 빈도(count)는 직접 제공되지 않으나 연도별 총 N과 퍼센트를 기반으로 계산 가능합니다 (e.g., count = (pct / 100) * valid N).

### 2. 변수별 분포
각 변수의 응답값별 퍼센트(pct)와 유효 N(결측 제외 기준)을 정리합니다. 퍼센트는 유효 응답자 기준으로 계산된 값입니다. 비해당(-1) 비율이 0%인 경우 유효 N = 총 N.

### 3. 결측/비해당 처리 규칙
- **결측 코드**: -8 (모르겠다/선택할 수 없음/무응답/DK/NA). 이 값은 분포 계산 시 제외 (유효 응답에서 % 산출).
- **비해당 코드**: -1 (문항 미실시 연도 또는 해당되지 않음). 이 값도 분포 계산 시 제외. 일부 변수(예: SATFIN, UNIFI)에서 특정 연도에 100% 비해당으로, 해당 연도 문항 미포함.
- **제외 규칙**: 퍼센트는 유효 응답자(1~5 범주) 기준. 총 N에서 -1과 -8 비율을 뺀 값이 유효 N. 무응답은 강제되지 않으며, 설문 설계상 "모르겠다" 옵션 제공.

### 4. 재코딩 규칙
- **CONFINAN 및 CONLEGIS**: 원 설문은 일부 연도 4점 척도(1=매우 신뢰, 2=다소 신뢰, 3=별로 신뢰하지 않음, 4=전혀 신뢰하지 않음)이나, 코드북 테이블은 3점으로 재코딩: 3(별로) + 4(전혀) = 3(거의 신뢰하지 않음). 이는 일관성 위해 적용된 것으로 보임.
- **기타 변수**: SATFIN, PARTYLR, NORTHWHO, UNIFI, KRPROUD는 원 척도 그대로 (재코딩 없음). PARTYLR은 5점, NORTHWHO 4점, UNIFI/KRPROUD 4점 + 결측.

### 5. 가중치 사용 여부
- 가중치 미사용. 모든 퍼센트는 비가중(unweighted)으로, 표본 크기(N) 기반 단순 비율. 코드북에 가중치(WT 등) 언급 없음. 만약 가중 필요 시 원 데이터셋(KGSS SAV/CSV)에서 별도 적용 추천 (e.g., 인구통계 가중).
